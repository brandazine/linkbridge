# Contibutor Guide
We are developing using pnpm. Please use pnpm.
Examples can be found in the `example` folder. You must run `example/native` and `sample/web` at the same time.
## Setup

```sh
$ pnpm i
$ pnpm build
```
## React Native (iOS) Start

```sh
$ cd example/native
$ pnpx pod-install
$ pnpm ios
```
## Web Start

```sh
$ cd example/web
$ pnpm dev
```
